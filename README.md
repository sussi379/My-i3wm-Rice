Pkgs that are needed - rofi, kitty, feh, polybar

1. Move the wallpaper.jpg to ~/Pictures
2. Move config to ~/.config/i3
3. Move config.ini to ~/.config/polybar/
4. Move battery.sh to ~/.config/polybar/scripts/
5. enter this command in terminal "sudo chmod +x battery.sh"
6. Move kitty.conf to ~/.config/kitty/
7. Move config.rasi to ~/.config/rofi/
